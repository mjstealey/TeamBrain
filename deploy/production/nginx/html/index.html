What is this?

TeamBrain is a multi-tenant, AI-agnostic shared memory service for the fabric-testbed/fabric-core-api pilot. Developers capture and search project context — decisions, gotchas, conventions — from Claude Code, Cursor, ChatGPT, gemini-cli, Copilot, or any MCP-aware tool. A parallel REST surface is available for non-MCP clients (GitHub Actions, OpenAI function calling, custom agents).

To join the pilot, sign in once with the GitHub account listed on the fabric-testbed/fabric-core-api repository. Within 15 minutes the membership sync picks you up and grants you a role. Source: fabric-testbed/TeamBrain.
